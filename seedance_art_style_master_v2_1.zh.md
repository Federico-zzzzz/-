# Seedance2.0 美术风格母版 v2.1

用途：本文件是“全局总控层”。  
每次正式分镜表前，必须先生成当前项目 / 当前场景专属的“美术风格母版”。

本文件负责四件事：

1. 统一全局画质、色彩、特效、声音与负向约束
2. 统一“全局提示词尾列”的写法，避免每镜重复写全局项
3. 定义男主 / 女主在新场景首次出场时的“轻微无逻辑逆光轮廓”规则
4. 规定每集结尾必须保留 `+2s 剪辑点`

## 动态生成规则

美术风格母版不能只复制默认模板，必须根据当前剧本动态改写以下项目：

1. 场景基调
2. 色彩系统
3. 角色级特效
4. 环境级特效
5. 声音风格
6. 主角识别
7. 主角新场景首次出场规则中的姓名占位

举例：
- 如果本段是现代直播厅对峙，就突出冷蓝、冷白、电子屏反光、硬光、死寂与舆论压迫
- 如果本段是国运战场，就突出暖金、逆光、沙尘、体积光、低频与宏大战场尺度
- 如果本段是豪门宴会或古代内殿，就突出暖烛光、木质深色、低饱和高对比、金色边缘反光
- 如果本段有巨物、神明、法相，就突出极低位仰拍、体量对比、巨型阴影、呼吸卷尘与低频震动

## 主角新场景首次出场逆光规则

这是硬规则。

### 触发时机
仅在以下条件同时满足时触发：
1. 当前角色是男主 / 女主 / 主角A / 主角B
2. 当前镜头是该角色在“新场景”中的首次出场镜头

### 实现目标
- 让人物边缘更有主角感
- 轻度增强出场仪式感
- 形成轻微朦胧感
- 不破坏原场景主光逻辑
- 不把整个镜头打成廉价光晕

### 写法要求
提示词里应写成：
- 人物背后出现轻微无逻辑逆光轮廓，沿发丝、耳廓、肩线与外轮廓低强度勾边，形成朦胧主角出场感，但不压过场内主光。
- 或：
- 背后补一层低强度轮廓逆光，轻轻勾亮发丝和肩线，制造若有若无的主角出场雾感。

### 禁止事项
- 不要把逆光写成高强度顶光
- 不要让逆光变成大片炫光
- 不要每个镜头重复写满
- 不要让逆光盖住人物五官
- 不要让逆光改变本来就确定的光位逻辑

## 如何避免每镜重复写全局项

从本版本开始：

1. 全局画质
2. 全局比例
3. 全局负向约束
4. 全局统一风格
5. 主角首次出场逆光规则

统一只写在两个地方：
- 美术风格母版
- 正式分镜表最后一列“全局提示词尾列”

### 每镜提示词只写什么
每镜 Seedance2.0 视频提示词只负责：
- 故事描述
- 本镜镜头执行
- 本镜人物动作
- 本镜表情与微动作
- 本镜声音 / 台词
- 本镜特效 / 光效变化
- 本镜专属补充约束（如果有）

这样可以减少冗余，并提高可读性与复制效率。

## 全局提示词尾列使用办法

在正式分镜表中：

1. 只在该集首个镜头的“全局提示词尾列”单元格中写完整内容
2. 该集后续镜头该列留空
3. 复制到 Excel / 飞书 / Google Sheets 后，从首镜到 `+2s 剪辑点` 行纵向合并并居中
4. 这样复制时不会丢失全局提示词，也不会让每镜提示词过长

## +2s 剪辑点执行规则

每集最后都必须追加一行 `+2s 剪辑点`。

执行原则：

- 这是编辑缓冲，不是新增剧情镜头
- 不能增加新台词
- 不能增加新动作结果
- 只能延续上一镜的空间、情绪、动作余韵或静止压迫
- 镜头可固定，也可极轻微延续上一镜运动
- 常用于：
  - 留悬念
  - 让BGM收尾
  - 卡点切下一集
  - 给观众留喘息与情绪余震

## 主角逆光写法示例

### 男主新场景首次出场示例
故事描述：男主第一次踏入新的空间，本镜先确立他的主角压场感；镜头执行：中近景，低位轻仰，缓慢推进；画面主体：男主站在门内半步位置，衣摆轻动；人物表演：他先沉默，随后抬眼，呼吸平稳，嘴角没有动作，手指自然下垂但带轻微张力；光效：人物背后补一层轻微无逻辑逆光轮廓，沿发丝、耳廓、肩线低强度勾边，形成若有若无的朦胧主角出场感，但不压过室内主光。

### 女主新场景首次出场示例
故事描述：女主第一次进入新的场景，本镜先立住她的气场与存在感；镜头执行：近景平视，轻微缓推；画面主体：女主从廊下步入暖光边界，裙摆轻扫地面；人物表演：她目光先落在前景，再慢慢抬起，呼吸很轻，嘴角没有笑意，左手手指微微收拢；光效：人物背后加入低强度轮廓逆光，轻轻勾亮发丝与肩线，制造轻雾感与主角出场质感，但不改变原有烛光主方向。
